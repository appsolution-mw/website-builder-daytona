## Scope

Provide a one-shot bootstrap script that provisions the Hetzner edge server
needed for the H.1c live validation. The script creates a Tailscale auth key,
spawns a Hetzner CX22 with cloud-init that installs Tailscale + Caddy
(Cloudflare DNS plugin), waits for the device to join the tailnet, sets the
Cloudflare wildcard DNS record, and appends `CADDY_ADMIN_URL` to `.env`.

## Plan

- Read tokens via `process.loadEnvFile(".env")` (Node 24 native) so no extra

## Notes

- Script is idempotent for the Cloudflare record (PUT on existing wildcard).
- Worker-Agent and Sandbox images still need to be built and pushed to a
  public registry before the validation flow can succeed; out of scope here.
- The host that runs the app must join the tailnet to reach Caddy admin on
