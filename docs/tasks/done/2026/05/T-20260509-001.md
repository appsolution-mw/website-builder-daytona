## Scope

Ersetzt die heutige Claude Code Integration (CLI-Subprozess via `claude --print
--output-format stream-json` aus dem Broker heraus) durch die offizielle
`@anthropic-ai/claude-agent-sdk`. Ziele:

- Token-Streaming via SDK `stream_event.text_delta` für interaktives Erlebnis.
- Robustes Session-Resume: SDK-JSONL primär, DB-Replay als Fallback.
- Attachments (Bilder/PDFs/Files) durchschleifen.
- Skills, Subagents, CLAUDE.md aus Host-Defaults gemerged mit Per-Projekt-Overrides.
- Permission-Gating via PreToolUse-Hooks (destruktive Patterns blockieren).
- Hard Cutover, ein PR.

## Plan

Implementierung wird auf parallele Subagents verteilt (Slices A–F, siehe Spec
§9). Reihenfolge: A/B/C/D parallel; E baut auf A; F integriert.

- A: agent-runner Fastify-Service mit SDK-Bindings, Bootstrap-Endpoint, In-Memory-Session-Map.
- B: Broker-Bridge (Mapping SDK→Protocol, neue Events).
- C: Host-Adapter + Replay-Context-Builder + Resume-Detection.
- D: agent-context/ Defaults (CLAUDE.md, 3–5 Skills, 2–3 Subagents).
- E: Hooks-Policy (Bash-Blocklist, Workspace-Boundary) + Tests.
- F: Prisma-Migration, Sandbox-Image-Cleanup, E2E-Smoke auf Staging.
