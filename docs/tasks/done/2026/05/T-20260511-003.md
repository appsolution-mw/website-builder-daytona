## Scope

Akkurate Cost- und Token-Erfassung wenn Claude Agent SDK über OpenRouter
läuft. OpenRouter strippt `total_cost_usd` aus den Anthropic-API-compat
Responses; Result: `TokenUsage.costUsd = 0` und das Phase-1.4e Quota-System
greift nicht. Agent-runner fängt `assistant`-Message-IDs ab und fetched
nach Turn-Ende die echten Kosten via OpenRouter Generation-API.

  Erstmal nur `claude-code`-Runtime, OpenHands separat.
